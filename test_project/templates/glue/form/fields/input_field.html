Documentation

Description

The Input Field is a basic form field that allows users to enter text data. It is implemented using the GlueCharField class which extends the GlueBaseFormField. This field is suitable for collecting short text inputs such as names, titles, or other single-line text data.

Special Considerations

  • For password input, use a separate password field type instead of this field
  • For multi-line text, use a text area field instead
  • The field automatically generates a label based on the field name if none is provided
