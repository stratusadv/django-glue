## Changes Implemented

### 1. Fixed proxy delete() Bug

**File**: `django_glue/proxies/queryset/proxy.py` (line 120)

**Problem**: The `delete()` method called `self._get_instance_by_pk(pk)` but the actual method is named `_get_model_instance_by_pk(pk)`. This caused an `AttributeError` at runtime when attempting to delete via QuerySet proxy.

**Fix**: Changed line 120 from:
```python
target_instance = self._get_instance_by_pk(pk)
```
To:
```python
target_instance = self._get_model_instance_by_pk(pk)
```
